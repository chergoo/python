# A股随机抽签器

用 Python + Flask 做的小工具：点一下按钮，随机抽出一只 A 股，展示它的代码、名称和实时行情（价格、涨跌幅、开高低、成交量、市值、市盈率等）。

## 运行方法

```bash
cd astock_random
pip install -r requirements.txt
python app.py
```

然后浏览器打开 http://127.0.0.1:5000

## 数据来源说明

- **正常联网**：启动时会通过 `akshare` 拉取全市场 A 股实时行情快照并缓存（5 分钟自动刷新一次），随机抽取时直接从真实数据里挑一只股票，信息真实可靠。
- **无法联网 / akshare 拉取失败**：会自动降级为"模拟数据"模式——按照真实的 A 股代码规则（沪市主板 600/601/603/605、科创板 688、深市主板 000/001、中小板 002/003、创业板 300/301、北交所等）随机生成一个"看起来真实"的代码，并配上合理范围内的模拟价格、成交量等信息。这种情况下界面会明确标注**"模拟数据"**，不会和真实行情混淆。

## 接口

| 路径 | 方法 | 说明 |
|---|---|---|
| `/` | GET | 页面 |
| `/api/random` | GET | 随机抽取一只股票，返回 JSON |
| `/api/refresh` | POST | 强制刷新行情缓存 |
